#  Scraping de Resultados UNMSM

##  Descripción
Este proyecto automatiza la extracción de resultados del examen de admisión de la Universidad Nacional Mayor de San Marcos (UNMSM) utilizando Python y Selenium.

El script recorre todas las carreras disponibles, accede a sus respectivos resultados y extrae la información completa de los postulantes, incluyendo la paginación dinámica (DataTables).

---

## Tecnologías utilizadas
- Python
- Selenium
- Pandas
- WebDriver Manager
